2. Configuration of input files

2.1. Overview

Input files are in csv format with header file.

Before the header there must be commented metadata for current file.

First not commented row must contain headers.

Rows after that will contain the data itself.

2.2. Metadata

In metadata there are two kinds of variables: Required and optional.

Variables must be at the beginning of the file and each row must start with hashtag. For comments add two hashtags.

Required values must be filled for application to understand what data is in that file.

Optional variables are for extra information.

Required variables

GENE_COLUMN - Header of the column from where application will get the genes.
ORGANISM - Organism name who genes are in given input file.
GENE_ID_TYPE - Name of the format in what genes in given input file are.

Optional variables

DEFAULT_TAG_HEADER - Header of column what should be selected by default when selecting given input file.
FILE_DESCRIPTION - Description of the given file what will be showed in tooltip in selection of the files.
DEFAULT_GENES - Example genes what will be added to the request when selecting given input file.
<HEADER_NAME> - You can write description for the header, what will be showed in tooltip in selection of the headers.

For <HEADER_NAME> variable example: if there is a header named "Disease", you can add description for that header as "#Disease=Possible Diseases" and that description will show up in tooltip.
